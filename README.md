# Netflix Dashboard – Power BI Project

## 📌 Overview
This project analyzes the Netflix Titles dataset using **Power BI** to uncover insights about content distribution, genres, and growth trends. It is beginner-friendly and requires no coding.

## 📂 Dataset
- Source: [Kaggle – Netflix Titles](https://www.kaggle.com/datasets/shivamb/netflix-shows)
- Format: CSV
- Columns: show_id, type, title, director, cast, country, date_added, release_year, rating, duration, listed_in, description

## ⚙️ Tools Used
- Microsoft Excel (data cleaning)
- Power BI (visualization)

## 📊 Dashboard Features
- Total Titles KPI
- Movies vs TV Shows distribution
- Genre analysis
- Country-wise content distribution (map visual)
- Trend of titles added per year

## 🚀 How to Run
1. Download dataset from Kaggle.
2. Clean data in Excel (optional).
3. Import CSV into Power BI.
4. Open `Netflix_Dashboard.pbix` file.
5. Explore interactive visuals.

## 📈 Insights
- USA contributes the highest number of titles.
- Drama and Comedy dominate genres.
- Netflix has steadily increased content since 2015.
- TV shows are gaining prominence compared to movies.
